## 2. g++ 命令

> g++ test.cpp -o test

+ 基本内容
  + g++ 指定编译器
  + ./test.cpp 源代码
+ 生成文件
  + -o 生成可执行文件
  + -c 生成目标文件
  + -s 生成汇编代码
  + -lm 链接数学库
  + --save-temps 保留临时文件
+ 其他参数
  + -std=c++0x 指定标准
    + c++0x 为 c++11
    + c++1y 为 c++14
    + c++1z 为 c++17
  + -wall 关闭编译时警告
